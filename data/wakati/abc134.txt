問題

数列 中 の を 除く 個 の 要素 の うち の 最大 の 値 を 求め よ 。

考察 
入力 が 大きい ので n 個 ある n - 1 個 の 要素 の 最大 値 を それぞれ 計算 し て い たら 間に合わ ない 
少し 考える と ， 必要 な 値 は n 個 （ 配列 全体 の ） 最大 値 と それ 以外 の うち の 最大 値 だけ と わかる ． 
以下 の 場合 分け を 考える 
N 個 の 数字 の 最大 値 と なる 要素 が 1 つ の 場合 : 最大 値 の index 以外 の 部分 数列 に対して は 2 番目 に 大きな 値 を 出力 ， それ 以外 の 部分 数列 に対して は 最大 値 を 出力 
N 個 の 数字 の 最大 値 と なる 要素 が 1 つ 以上 ある 場合 : 全て の 部分 数列 に対して 最大 値 を 出力
