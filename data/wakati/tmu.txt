大学院 に 合格 し た 
久しぶり の 投稿 です ． 先日 大学院 試験 の 結果 が 出 て ， 無事 東京都立大学 の 小町 研 に 合格 し た ので 受験 体験 記 的 な もの を 書き たい と 思い ます ． 
経緯 
大学 は 経済学部 な の です が 主 に 統計 の 勉強 を し て い た ため ， 機械 学習 と の 親和 性 も 高かっ た です ． 機械 学習 に対して 興味 を 持っ た の は 友人 の 影響 で ， 友人 に 影響 を 受け て 始め て 以来 独学 で 機械 学習 周り の 勉強 を し て き まし た ． 
また ， 自然 言語 処理 に対して 興味 を 持っ た 決定的 な 要因 は アルバイト です ． 私 は 1 年 半 前 くらい から フューチャー 1 で 働い て いる の です が ， そこで 自然 言語 処理 系 の プロジェクト に アサイン さ れ て ， 本格 的 に 自然 言語 処理 に 触れる 機会 が でき まし た ． アルバイト で の 経験 が 自分 の 自然 言語 処理 に対する 興味 を 育て た と 思っ て い ます ． 
志願 まで の 過程 
小町 研 の HP 2 に も ある 通り ， 志願 前 に 小町 先生 に 英語 の スコア の 報告 と 研究 計画 書 を 提出 し て 受験 の 許可 を もらわ ない と いけ ませ ん ． 僕 は 4 月頃 に 小町 先生 に 連絡 し まし た （ ギリギリ だっ た みたい です ） ． 受験 の 意思 を 確認 し た 後 ， 小町 先生 と オンライン で 面談 を し まし た ． ここ で は 研究 室 と 自分 の 研究 し たい 内容 の マッチング ， また 受験 許可 が 降りれ ば ， 入学 まで 自然 言語 処理 | 機械 学習 の 基礎 勉強 会 に 参加 する こと に なる ので ， そちら の 参加 意思 の 確認 と スケジューリング を 行い まし た ． 
また ， 上述 の 基礎 勉強 会 に も 参加 し まし た ． 僕 は アルバイト の 都合 で ， 奥村 先生 の 「 言語 処理 の ため の 機械 学習 入門 」 を 教材 と し た 機械 学習 勉強 会 と ， NAIST Graham Neubig 先生 の 教材 3 を 使っ た プログラミング 演習 の 2 つ に 参加 し まし た ． 入学 前 に こう いっ た 機会 を 設け て いただい た おかげ で ， 研究 室 の 雰囲気 を 掴む の に 役立つ と 同時に ， 受験 勉強 の モチベーション に も なり まし た ． 
小町 研 を 受験 する ため に 必要 な の は 以上 の こと くらい で ， あと は 都立 大 の ほう に 出願 書類 を 送っ て 受験 の 準備 は 完了 です ． 
対策 
小町 研 は 都立 大 の システム デザイン 研究 科 情報 科学 域 に あり ， 情報 科学 域 の 受験 科目 は 数学 ， 英語 ， 面接 の 3 科目 です ． うち 英語 は 外部 スコア の 提出 と 決め られ て おり ， TOEIC か TOEFL iBT ， 今年 は それら に 加え て duolingo english test が 含ま れ て い まし た ． 
数学 について は ， 小町 先生 に コンタクト を とっ た あたり ， 受験 の 4 ヶ月 前 くらい から 始め ， 加藤 文 元 先生 の チャート 式微 分 積分 | 線形 代数 を 2 周 ほど し まし た ． それ に 加え て 東大 の 微 積 線形 の 授業 スライド を 使っ て ゼミ の 友人 と 一緒 に 輪読 を し て 理解 を 深め まし た ． 個人 的 に は ， 私 が 経済学部 で 理系 の 大学 数学 に対して の 理解 が 深く ない こと も あり ， 疑問 点 の 解消 や 理解 を 深める こと が できる という 点 で ， 後者 の 輪読 形式 の 勉強 が 非常 に 役に立ち まし た ． 
英語 で は ， 私 は TOEIC 705 点 ， TOEFL 74 点 （ 有効 期限切れ ） しか 持っ て い なかっ た ため ， 改めて スコア を 取得 する 必要 が あり 苦労 し まし た ． 数学 の 勉強 が 落ち着い た 6 月頃 に 英語 の 対策 も 開始 し ， 7 月 に iBT と duolingo の 受験 予約 を し て ， iBT 対策 を 中心 に 勉強 を 進め まし た ． 私 は speaking が 苦手 だっ た ので ， DMM 英会話 に 入会 し ， 毎日 スピーキング の 対策 を し まし た ． 結果 ， iBT は 得意 の reading と writing で こけ て しまい 3 年 前 の スコア と と 変わら ず 74 点 ， duolingo で は 110 点 （ iBT 換算 で 90 点 相当 ） を とっ て duolingo の ほう を 提出 し まし た ． 
面接 は 小町 先生 曰く あまり 合否 に 直結 し ない らしく ， 面談 に 近い と 聞い て い た ので 特に 対策 は 行い ませ ん でし た ． 一応 ， 前日 に 研究 計画 書 について 言及 さ れる と 想定 し て ， 研究 計画 書 を 確認 し まし た が ， 特に 言及 は あり ませ ん でし た ． 
受験 と 合格 発表 
数学 は 例年 より 簡単 な 印象 を 受け まし た ． 積分 の 問題 に 癖 が あり ， 一 問 解け ませ ん でし た が それ 以外 は 大抵 解け まし た ． 例年 だ と 微分 方程式 あたり まで は 試験 範囲 という 印象 を 持っ て い まし た が ， 今年 は 高校 数学 の 微分 積分 で とける 問題 ばかり だっ た 気 が し ます ． 線形 代数 について も 行列 の 対 角 化 まで しか 試験 問題 に なっ て おら ず ， 一部 証明 問題 を のぞき ， 計算 ミス と の 戦い という 印象 を 受け まし た ． 
面接 は 小町 先生 を 含め て 5 人 の 先生 と 面接 ． 上述 の 通り ， 研究 内容 について の 言及 は なく ， 合格 し た 際 の 入学 意思 と 抱負 を 話し ， 5 〜 10 分 で 終了 し まし た ． 
結果 は 2 週間 ほど で 出 まし た ． 合格 発表 の 日 は 旅行 中 だっ た ので 不 合格 で ムード を 壊す よう な こと に なら ず よかっ た です ． 
おわり に 
自然 言語 処理 を 研究 できる 研究 室 に 合格 でき て 嬉しい です ． 応援 し て くださっ た 方 ， お祝い し て くださっ た 方 ありがとう ござい まし た ． 
10 月 から 早速 小町 研 に join し て 研究 が 始まる ので 頑張り たい と 思い ます ．
